# Async Console Game (curses)

Консольная мини-игра на Python: звёздное небо и анимированный корабль с управлением.

---

## Требования

- Python 3.10+ (рекомендуется 3.11)

### Если Windows
На Windows модуль curses отсутствуетю Пропишите команду:
```bash
pip install windows-curses
```

## Запуск

Из корня проекта
```bash
python main.py
```
Остановка программы: Ctrl + C.

## Управление
- Корабль перемещается по игровому полю с помощью стрелок на клавиатуре
