# Async Console Game (curses)

Консольная мини-игра на Python: звёздное небо и анимированный корабль с управлением.

---

## Требования

- Python 3.10+ (рекомендуется 3.11)

### Если Windows
На Windows модуль curses отсутствуетю Пропишите команду:
```bash
pip install windows-curses
```

## Запуск

Из корня проекта
```bash
python main.py
```
Остановка программы: Ctrl + C.

## Управление
- Корабль перемещается по игровому полю с помощью стрелок на клавиатуре

## Ход игры

Игра начинается в спокойном режиме: на экране видно звёзды и корабль, которым можно управлять стрелками.

Дальше время в игре идёт вперёд — внизу экрана отображается год и короткие события.  
По мере продвижения игра становится сложнее: в космосе появляется всё больше мусора (камней/обломков), и вам нужно уворачиваться от них.

### Важный момент
- До 2020 года корабль не может стрелять — можно только маневрировать и избегать столкновений.
- С 2020 года открывается стрельба (`Space`): теперь можно уничтожать мусор выстрелами.

Если выстрел попадает в мусор:
- мусор исчезает,
- появляется анимация взрыва

Если корабль сталкивается с мусором:
- игра заканчивается,
- на экране появляется Game Over

### Цель игрока
Продержаться как можно дольше, уворачиваться от мусора и (после 2020 года) расчищать путь выстрелами.
